## Usage:

Running the following command will generate the LP files, process the CPLEX output, produce a CSV file containing key data, and save the graphs. The report can be made seperately using a LaTeX editor.

### Windows (7 or later):

```
output.ps1
```

### Linux:

```
output.sh
```
